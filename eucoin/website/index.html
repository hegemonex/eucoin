What is TCoin?

TCoin (TC) is a fully compliant ERC-20 token deployed on Ethereum Sepolia. Built with Solidity and Hardhat, it demonstrates core blockchain principles — transparency, immutability, and decentralisation.

🔒

Secure

Built using OpenZeppelin's audited ERC-20 contracts.

🌍

Global

Anyone with an Ethereum wallet can hold and transfer TCoin.

⚡

Fast

Transactions settle in seconds with minimal gas fees.

📖

Transparent

All transactions are publicly visible on the blockchain.
